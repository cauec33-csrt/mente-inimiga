# mente-inimiga
Sua mente tem um inimigo. E esse inimigo é ela mesma. O Método Mente Inimiga descobre qual dos 4 modos de operação está ativo no seu caso — antecipação, loop, julgamento ou sobrecarga — e te entrega o protocolo exato para desativá-lo em 7 dias. Sem enrolação. Só o que funciona. 🛡️ 7 dias de garantia.
